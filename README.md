# AI Sales Intelligence Dashboard

An interactive **data analytics dashboard** built with **Python, Flask, Pandas, and Chart.js** that analyzes sales data, generates insights, predicts future sales using machine learning, and answers natural language queries.

---

## 🚀 Features

* 📊 **Interactive Charts**

  * Sales by Product (Bar Chart)
  * Sales by Region (Pie Chart)
  * Monthly Sales Trend (Line Chart)

* 🤖 **AI Data Assistant**

  * Ask questions about sales data
  * Example: *"top product"*

* 📈 **Sales Prediction**

  * Predict future sales using **Machine Learning**

* 📉 **Business Insights**

  * Top region
  * Best selling product
  * Total revenue
  * Average sales value

* 🔎 **Region Filtering**

  * Analyze sales by specific regions

## ▶ How to Run the Project

### 1️⃣ Clone the repository

```
git clone https://github.com/Rakesh0788/sales-ai-dashboard.git
```

### 2️⃣ Install dependencies

```

```

### 3️⃣ Run the application

```
python app.py
```

### 4️⃣ Open in browser

```
http://127.0.0.1:5000
```
